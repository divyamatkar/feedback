Design and Development of Login Page

Aim:

To design and develop a responsive Login Page using HTML, CSS, and Bootstrap that allows users to securely enter their email and password, with options for “Remember Me” and “Forgot Password?” to access the Dashboard.

Objectives:

  • To create a user-friendly login interface using HTML form elements.
  • To apply CSS and Bootstrap classes for layout, styling, and responsiveness.
  • To understand the use of form controls like input fields, checkboxes, and buttons.
  • To implement UI features such as “Remember Me” and “Forgot Password?” for better usability.

Theory:

HTML (HyperText Markup Language):

  • Used to define the structure of the web page.
  • <form>, <input>, <label>, and <button> tags are used to create the login form.
  • Input fields collect user credentials like email and password.

CSS (Cascading Style Sheets):

  • Used to style the login page and make it visually appealing.
  • Properties like background-color, padding, margin, border-radius, and box-shadow are used to design the card layout.

Bootstrap:

  • A popular front-end framework that provides pre-defined CSS classes and components.
  • Classes like container, form-control, btn, btn-primary, mb-3, d-flex, justify-content-between, and text-center help in quick layout creation and responsiveness.
  • Makes the page responsive so it looks proper on different screen sizes (desktop, tablet, mobile).

Login Page Concept:

  • A login page is used to authenticate users before giving access to a protected area like a Dashboard.
  • It typically includes:
    • Email field
    • Password field
    • “Remember Me” checkbox (for saving login preference)
    • “Forgot Password?” link (for password recovery).

Conclusion:

By designing and developing the login page using HTML, CSS, and Bootstrap, we learned how to create a responsive and user-friendly authentication interface. This practical helped us understand form creation, styling with CSS, use of Bootstrap components, and the basic structure of a real-world login system used before accessing a Dashboard.

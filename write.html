Program: Area of Triangle using JavaScript

Aim: To develop a JavaScript program that calculates the area of a triangle and displays the result to the user.

Objective:

  • To understand how to accept user input using HTML form controls.
  • To apply mathematical operations in JavaScript.
  • To use JavaScript functions and DOM manipulation to display output dynamically on a webpage.

Theory: A triangle is a polygon with three sides. The mathematical formula used to calculate the area of a triangle is:

Area = ½ × Base × Height

Where:

  • Base (b): The bottom side of the triangle
  • Height (h): The perpendicular distance from the base to the opposite vertex

Using JavaScript, user values can be captured using document.getElementById(). The calculation is then performed in a function, and the result is displayed back on the webpage using DOM manipulation.

Conclusion:

By performing this program, we successfully learned how to calculate and display the area of a triangle using JavaScript. This practical helped in understanding how to take user input, perform mathematical operations, and dynamically update the webpage using the DOM.

Version Control with GitHub

Steps to Create a Version Control Account on GitHub

  1. Open the website: github.com
  2. Click Sign Up.
  3. Enter your email, password, and username.
  4. Verify your email.
  5. Login to your GitHub account — setup is complete.

Steps to Create a New Repository on GitHub

  1. Click New Repository.
  2. Enter a repository name.
  3. Choose Public or Private.
  4. Click Create Repository.

Steps to Use Git Commands to Push Code to GitHub

  1. Open the folder where your project is saved.
  2. Right-click and open Git Bash.
  3. Run the following commands:
       git init
       git add .
       git commit -m "First commit"
       git branch -M main
       git remote add origin https://github.com/your-username/repository-name.git
       git push -u origin main

Now your code is uploaded to GitHub successfully.
